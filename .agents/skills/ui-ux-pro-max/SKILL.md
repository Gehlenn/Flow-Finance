# UI/UX Pro Max - Flow Finance Edition

## Descrição
Skill de design e UX optimization. Foco em simplificação, hierarchy, composition restrained, e consistency visual. Alinhado com prioridade de produto: simplificar Flow em torno de fluxo de caixa inteligente.

## Quando Usar
- Auditoria visual de componentes (consistency, spacing, hierarchy)
- Simplificação de navegação e dashboard
- Ocultação de features complexas (Open Finance, Pluggy)
- Design do fluxo de assistente consultivo
- Microcopy e tone of voice
- Landing pages e marketing UI

## Comandos Principais
```bash
/design-review          # Auditoria visual de site/app em produção
/design-shotgun         # Exploração de variantes de design
/design-consultation    # Criação de design system do zero
/page-cro              # Otimização de conversão em páginas
/design-html           # Implementação de designs aprovados
```

## Integração com Flow Finance
- Alinha com direção: simplificação UI + remoção Open Finance/Pluggy
- Melhora readability de transações e saldo
- Consultor IA comunicado como apoio prático (não autonomia ampla)
- Hierarquia: Caixa → Transações → Receitas → Assistente → Config

## Princípios Aplicados
- ✅ Restrained composition (foco, sem clutter)
- ✅ Image-led hierarchy (visual scanning eficiente)
- ✅ Cohesive content structure (padrão consistente)
- ✅ Tasteful motion (não gimmicks)
- ❌ Evitar: generic cards, weak branding, UI clutter
