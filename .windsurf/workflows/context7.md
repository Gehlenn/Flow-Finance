# Context7 Workflow

## Overview
Context7 provides semantic memory and context persistence across sessions.

## 1. Memory Operations
- Store important context using create_memory tool
- Tag memories with project and category
- Link related memories for discovery

## 2. Context Retrieval
- Use trajectory_search for conversation history
- Query memories by tags and content
- Cross-reference with code_search

## 3. Vault Integration
- Sync with Obsidian vault at: `E:\app e jogos criados\obsidian-vault`
- Use Graphify for knowledge graph visualization

## Usage
Memories are automatically retrieved at conversation start.
