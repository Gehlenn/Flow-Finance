# GSD Workflow - Adapted for MMORPG & Flow Finance

## Quick Commands

### Planning
- `/gsd-discuss-phase` - Discuss phase requirements
- `/gsd-plan-phase` - Create detailed phase plan
- `/gsd-analyze-dependencies` - Check phase dependencies

### Execution
- `/gsd-execute-phase` - Execute all plans in a phase
- `/gsd-autonomous` - Run phases automatically
- `/gsd-quick` - Fast task execution

### Quality
- `/gsd-verify-work` - UAT verification
- `/gsd-code-review` - Review changed files
- `/gsd-audit-fix` - Auto-fix issues

### Progress
- `/gsd-progress` - Check project status
- `/gsd-stats` - Show project statistics
- `/gsd-check-todos` - List pending tasks

## Project-Specific Workflows
