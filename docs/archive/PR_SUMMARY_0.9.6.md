# Resumo de PR - 0.9.6

## Papel deste documento

Este material registra o resumo da PR referente ao ciclo `0.9.6`, com foco em simplificação de interface e reposicionamento do produto.

## Resumo executivo

O objetivo central desta PR foi simplificar o Flow Finance em torno do foco real do produto:

- fluxo de caixa
- operação financeira
- leitura consultiva por IA

Principais efeitos descritos:

- navegação principal enxugada
- dashboard com indicadores de decisão
- ajuste de tom do assistente e do consultor IA
- centralização de microcopy
